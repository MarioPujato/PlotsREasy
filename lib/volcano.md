#### DATA FORMAT:
Data must be in the following format:

```
name    -log2(FC) -log(P)
AADACL2  2.12      2.6
AGO4     4.35      6.4
BIRC3    0.45      0.3
DDX26B   0.89      1.5
```

Each field must be tab separated and contain a header line.  
Only up to 50 highlighted genes can be labeled in the plot.
